# RayTracing
Implementation of ray tracing rendering algorithm

Scene setup can be made in MainViewModel.cs source file.

Contains following geometric primititives:
- Sphere
- Box
- Disk
- Plane
- Torus
- Quadric surfaces:
  - Ellipsoid
  - Cone
  - Cylinder
  - Elliptic paraboloid
  - Hyperbolic paraboloid
  - Elliptic hyperboloid
